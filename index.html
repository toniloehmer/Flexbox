Einleitung Geschichte Beispiele Warum Flexbox?

display

'display' ist keine für flexbox spezifische Eigenschaft, sondern eine CSS-Eigenschaft. Im Allgemeinen wird der Typ der Renderbox definiert, der für ein Element verwendet wird. Im Falle von Flexbox definiert es einen Flex-Container. Inline oder Block abhängig vom angegebenen Wert. Dies ermöglicht einen flexiblen Kontext für alle direkten Kinder.

.container {
  display: flex;
}

flex-direction

Legt die Richtung fest, in der Flex-Artikel im Flex-Container platziert werden. Flexbox ist ein Layoutkonzept in einer Richtung. Flex-Elemente haben hauptsächlich horizontale Zeilen oder vertikale Spalten. Die vier Optionen für die Flex-Richtung sind Zeile | Zeilenumkehrung | Spalte | Spalte umgekehrt. Zeile und Zeile umkehren sind abhängig vom Schreibmodus.

.container {
  flex-direction: row | row-reverse | column | column-reverse;
}

flex-wrap

Legt die Richtung fest, in der Flex-Artikel im Flex-Container platziert werden. Flexbox ist ein Layoutkonzept in einer Richtung. Flex-Elemente haben hauptsächlich horizontale Zeilen oder vertikale Spalten. Die vier Optionen für die Flex-Richtung sind Zeile | Zeilenumkehrung | Spalte | Spalte umgekehrt. Zeile und Zeile umkehren sind abhängig vom Schreibmodus.

  flex-wrap: nowrap | wrap | wrap-reverse;
}

Steuert die Reihenfolge, in der Flex-Elemente im Flex-Container angezeigt werden.

  order: ; /* default is 0 */
}

Steuert die Reihenfolge, in der Flex-Elemente im Flex-Container angezeigt werden.

  flex: none | [ <'flex-grow'> <'flex-shrink'>? |
